# PulseVote

*Fast opinions. One link. Real insight.*

PulseVote is a lightweight web application designed to collect opinions, votes, and ratings in the simplest way possible — through a single shareable link.

The project focuses on **clarity, speed, and elegant user experience**, with full support for Arabic (RTL) and English.

# What is PulseVote?

PulseVote allows anyone to create a poll or rating in minutes and share it instantly. Participants can vote without registration, making the process frictionless and accessible to everyone.

The app is suitable for:

* Event feedback
* Public opinion polls
* Quick decision-making
* Content creators and teams

# Key Features

* Single shareable link per poll
* No registration required for participants
* Arabic (RTL) and English support
* Works on all devices (mobile, tablet, desktop)
* Progressive Web App (PWA) – installable
* Instant results view after voting
* Anonymous voting
* One vote per device

* Supabase persistence (Postgres)

# How It Works

1. Organizer creates a poll or rating
2. Chooses question type (Yes/No, stars, options, emoji)
3. Shares one link
4. Participants vote instantly
5. Results are shown instantly after voting

# Tech Stack

* Single-file web app (HTML/CSS/JavaScript)
* Supabase (Postgres) for data persistence
* Progressive Web App (PWA) ready (installable when hosted on HTTPS)
